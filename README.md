# Vanvrindha

A luxury botanical workspace platform built using Django and MySQL.

## Features

- User Authentication
- Product Management
- Cart System
- Order Management
- Category Filtering
- Search Functionality
- Admin Controls
- Luxury Botanical UI

## Tech Stack

- Python
- Django
- MySQL
- HTML
- CSS
- JavaScript

## Architecture

- Repository Pattern
- Service Layer
- Layered Architecture
